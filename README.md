# Drone Hunt

Drone Hunt is an arcade shooting gallery inspired by classics like Crazy Chicken (Moorhuhn), Duck Hunt, and Barker Bill's Trick Shooting.

## Core Philosophy

The game unfolds the longer you play.

The unfolding is the knowledge gained.

Nothing is random.

Knowledge is progression.

Every object can matter.

Every choice has an opportunity cost.

## Features

- Fast arcade gameplay
- Multiple maps
- Hidden routes
- Environmental secrets
- Steam leaderboards
- Score attack
- Deterministic world
- Mouse and light gun support

## Engine

Godot 4
